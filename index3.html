Buatlah reading report mengenai perbedaan website dengan web application!

Website mempunyai tujuan utama untuk menyampaikan sebuah informasi kepada penggunanya. Di dalam website tidak terdapat fitur tambahan seperti di web apk untuk berinteraksi, baik dengan admin atau dengan pengunjung web yang lain. website bersifat informatif sedangkan web apk bersifat interaktif.Website application mempunyai beberapa fitur tambahan yang dapat digunakan para pengunjung untuk berinteraksi di dalam sebuah website. Berbeda dengan website, web apk lebih responsif terhadap tindakan pengguna nya. Website application dapat memberikan info yang lebih rinci serta membantu aktivitas pengguna di dalam website.

Jelaskan perbedaan Front End, Back End & Full Stack!

Perbedaan front end dengan back end developer adalah yang masuk dalam bagian front-end yaitu tampilan website yang dilihat oleh user. sedangkan back end pekerjaan ini sifatnya 'di balik layar'. Meski back end cenderung tak terlihat,tetapi tanpa back-end yang baik,kita tak bisa menikmati tampilan website yang menarik. Sedangkan full-stack developer berfungsi dalam pembangunan sebuah website, untuk menjalankan peranan lintas fungsional.
